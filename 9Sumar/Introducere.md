# Operatori
* operatori logici
```
    &&, ||
    >,<,=,!
```

* operatori aritmetici
```
    +,-,=,*,/,%
```
    
* incrementare/ decrementare
```
    ++x - preincrementare - incrementeaza si dupa ia valoarea
    x++ - postincrementare - ia valoarea si dupa incrementeaza 
```
> ex: x=5; y=3; y=x++ => y=5, x=6
<br>
> ex: x=5; y=3; y=++x => y=6, x=6

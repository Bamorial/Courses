# Command Line
## Afisare fisiere din directorul curent
```bash
ls
```
>linux / macOS
```bash
dir
```
>Windows

## Navigatie
- inainte
```bash
cd folder_name
cd ~/calea/spre/directorul/dorit
```
- inapoi
```bash
cd ..
```
## Creare de folder 
```bash
mkdir nume_folder
```
## Stergere folder
```bash
rmdir nume_folder
```
>pentru foldere goale
```bash
rm -r nume_folder
```
>pentru foldere cu fisiere/foldere in ele

## Fisiere

### Creare
```bash
touch fisier.extensie
```
### Stergere
```bash
rm nume_fisier.extensie
```
### Citire
```bash
cat nume_fisier.extensie
```


## C++
### Creare fisier
```bash
touch nume_fisier.cpp
```
### Edirare fisier
```bash
code nume_fisier.cpp
```
### Compilare fisier
```bash
g++ nume_fisier.cpp
```
> dupa compilare se genereaza fisierul a.out
### Rulare program
```bash
./a.out
```
